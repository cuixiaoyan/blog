1.valueOf：方法用于返回给定参数的原生 Number 对象值。 如：char[] str = { ‘a’, ‘b’, ‘c’, ‘a’ }; String s = String.valueOf(str); // s的值是 abca 2.indexOf：返回指定字符在字符串中第一次出现处的索引，如果此字符串中没有这样的字符，则返回 -1。 如：s.indexOf(“b”); // 表示：如果字符数组中存在b，则返回第一次出现b的下标索引，如果没有b，则返回-1
